# Assessment & Grading Service

## Overview
The Assessment & Grading Service is responsible for managing the entire lifecycle of student assessments. This includes creating exams, recording grades, and generating report cards for the Education Management System.

## Key Features
- **Exam Management**: Schedule and configure exams and quizzes.
- **Gradebook**: A digital gradebook for teachers to input and calculate grades.
- **Report Cards**: Automated generation of student performance reports.
- **Analytics**: Analyze class and student performance trends.
