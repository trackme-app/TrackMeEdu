# Scheduling & Timetable Service

## Overview
The Scheduling Service solves the complex constraint satisfaction problem of assigning classes, teachers, and students to rooms and times within the Education Management System.

## Key Features
- **Class Scheduling**: Automated and manual generation of class timetables.
- **Resource Booking**: Manage bookings for labs, lecture halls, and equipment.
- **Exam Timetables**: Coordinate non-conflicting exam schedules.
- **Calendar Integration**: Export schedules to standard calendar formats (iCal, etc.).
