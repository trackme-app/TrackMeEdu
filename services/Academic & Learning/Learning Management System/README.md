# Learning Management System (LMS) Service

## Overview
The LMS Service provides the digital interface for the learning experience. It connects students with course materials and facilitates assignments within the Education Management System.

## Key Features
- **Content Delivery**: Host and stream lectures, slides, and documents.
- **Assignment Submission**: Allow students to upload work for grading.
- **Discussion Forums**: Facilitate class discussions and peer interaction.
- **Interactive Quizzes**: Run automated online quizzes for immediate feedback.
