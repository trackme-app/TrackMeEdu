# Course & Curriculum Service

## Overview
This service handles the structural definitions of what is taught. It manages the catalog of courses, curriculum standards, and educational programs within the Education Management System.

## Key Features
- **Course Catalog**: Maintain a list of available courses and modules.
- **Syllabus Management**: Define topics, learning outcomes, and resources for each course.
- **Prerequisites**: Manage course dependencies and progression rules.
- **Program Management**: Define degree or certification requirements.
