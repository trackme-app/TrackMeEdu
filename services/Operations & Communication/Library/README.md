# Library Service

## Overview
The Library Service manages the institution's physical and digital resources within the Education Management System. It facilitates cataloging, lending, and tracking of library assets.

## Key Features
- **Catalog Management**: Database of books, journals, and media.
- **Circulation**: Check-in and check-out processes for borrowers.
- **Reservations**: Allow students to reserve items.
- **Digital Library**: Access to e-books and research papers.
