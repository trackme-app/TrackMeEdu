# Notifications Service

## Overview
The Notifications Service acts as the central communication hub for the Education Management System. It abstracts the complexity of sending messages across multiple channels.

## Key Features
- **Multi-channel Delivery**: Support for Email, SMS, Push Notifications, and In-app alerts.
- **Templates**: Manage notification templates for consistent branding.
- **Batch Processing**: Send mass announcements efficiently.
- **Preferences**: Manage user communication preferences.
