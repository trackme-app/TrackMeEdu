# Student Information Service

## Overview
The SIS is the central registry for all student data. It acts as the source of truth for student demographics and academic history in the Education Management System.

## Key Features
- **Student Profiles**: Store detailed biographic and demographic information.
- **Academic History**: Record past enrollments and achievements.
- **Guardian Integration**: Link students to parent/guardian contacts.
- **Document Repository**: Store transcripts and other student-related files.
