# Identity & Access Management (IAM) Service

## Overview
This service is the security cornerstone of the Education Management System, managing user identities, authentication, and authorization policies.

## Key Features
- **Authentication**: Secure login via SSO, MFA, or standard credentials.
- **Role-Based Access Control (RBAC)**: Define permissions for Students, Teachers, Admins, and Parents.
- **User Provisioning**: Automated creation and deactivation of user accounts.
- **Audit Logging**: Track access and security events.
