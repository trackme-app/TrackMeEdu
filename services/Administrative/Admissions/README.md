# Admissions Service

## Overview
The Admissions Service streamlines the intake process for new students. It manages inquiries, applications, and enrollment workflows for the Education Management System.

## Key Features
- **Application Portal**: Online submission of student applications.
- **Workflow Management**: Track application status through review stages.
- **Document Verification**: Manage uploads and verification of required documents.
- **Enrollment**: Finalize student admission and generate matriculation numbers.
