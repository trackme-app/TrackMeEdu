# Administrative Domain

## Overview
This domain manages the organizational and bureaucratic aspects of the institution. It serves as the backbone for data management regarding people and access control.

## Services
- **Admissions**: Manages the application process, student intake, and enrollment.
- **Identity & Access Management**: Handles user authentication, authorization, and role management across the platform.
- **Staff Information**: Maintains records for faculty and administrative staff.
- **Student Information**: Serves as the central repository for student profiles and demographic data.
