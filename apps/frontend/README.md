# Education Management System - Frontend

## Overview
This is the primary user interface for the Education Management System, built with **React**, **TypeScript**, and **Vite**. It provides a modern, responsive experience for students, faculty, and administrative staff to interact with the various backend services.

## Key Features
- **Role-Based Dashboards**: Tailored interfaces for Students, Teachers, and Administrators.
- **Micro-Frontend Ready**: Designed to integrate features from various educational domains (Learning, Admin, Operations).
- **Responsive Design**: Optimized for both desktop and mobile usage in educational settings.
- **Real-time Updates**: Integrated with backend services for live notifications and schedule changes.

## Tech Stack
- **Framework**: React 18+
- **Build Tool**: Vite
- **Language**: TypeScript
- **State Management**: Context API
- **Styling**: Vanilla CSS + styled-components

## Architecture
The frontend follows a modular structure where components are grouped by domain (e.g., Admissions, Course Management, Grading). It communicates exclusively with the **API Gateway** to interact with the backend microservices.
